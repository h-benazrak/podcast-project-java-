Podcast Scheduler Project
------------------------

Author:HANA SHAIMI 
ID : 22312235

Instructions to run:
1. Open the project in your IDE (Eclipse, IntelliJ, NetBeans, etc.).
2. Run the file: app.PodcastSchedulerApp
3. Use the GUI to create, schedule, publish, and save episodes.

Known Limitations:
- The app does not validate invalid time formats (like 25:00).
- Cannot edit or delete episodes (optional future improvement).

Extra Features:
- Buttons are styled with colors (CSS).
- Episodes automatically load from file when the app starts.
- Schedule conflicts are checked, and episodes in the past cannot be scheduled.
